2 Quantitative skills laboratory

PURPOSE

  1. To learn how to record data in electronic format
  2. To learn how to write hypotheses as equations
  3. To learn how to choose the appropriate visualizations
  4. To learn how to make graphs using R Studio

Before coming to the laboratory:

  1. If you have not already, install R and RStudio.

  2. Before coming to the laboratory read:

  • Introduction to R
  • Making graphs in R
  • Entering and loading data
